# Security Policy

## Supported Versions

We currently only support the latest version of Jeopardy Studio.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue. Instead, report it privately to the maintainers at [xMinhx@users.noreply.github.com].

We take security seriously and will respond as quickly as possible.
