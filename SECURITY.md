# Security Policy

## Supported Versions

We currently only support the latest version of Jeopardy Studio.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue. 

Instead, please use the **"Report a vulnerability"** button located in the [**Security**](https://github.com/xMinhx/jeopardy-studio/security) tab of this repository. 
This allows you to report the issue privately to the maintainers.

We take security seriously and will respond to your report as quickly as possible.
