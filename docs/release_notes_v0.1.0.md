## 🎬 Jeopardy Studio v0.1.0 (Pre-release)

Welcome to the first public pre-release of **Jeopardy Studio**!

This version marks our transition from a private university tool to an open-source project. Built for reliability and a professional "broadcast" feel, this release includes all core features needed to host a live quiz show.

### ✨ Key Features
*   **Dual-Monitor Sync**: Independent Host Control and Audience Display windows using Electron IPC.
*   **Live Scoreboard**: Real-time team management and automated score tracking.
*   **Offline-First**: Runs entirely locally—no internet connection required during your event.
*   **Daily Double & Final Jeopardy**: Full support for classic round variations and wagers.
*   **Timer & Audio**: Studio-quality countdowns and SFX triggers.

### 🗺️ The Roadmap
We are prioritizing core stability and visual excellence:
*   **Responsive Design**: Optimization for various screen aspect ratios and projector resolutions.
*   **Custom Themes**: Personalized branding, colors, and typography for the audience display.
*   **Internationalization**: Support for multiple languages.
*   **Advanced Animations**: Smoother, more cinematic transitions.
